# Family Task Management Application

A full-stack task management application for families.

## Technologies Used
- **Backend**: .NET Core Web API
- **Frontend**: Angular 19
- **Architecture**: 3-Layer Architecture (API, Core, Infrastructure)

## Project Structure
- `FamilyTask.API` - Web API layer
- `FamilyTask.Core` - Business logic and domain models
- `FamilyTask.Infrastructure` - Data access and external services
- `family-task-app` - Angular 19 frontend application
